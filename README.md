# PelisReact 

Aplicación web desarrollada con React que permite buscar películas y ver información detallada consumiendo la API de The Movie Database (TMDB).

El proyecto fue creado como práctica de frontend, enfocándose en el manejo de estados, consumo de APIs y buenas prácticas de UI/UX.

---

## Demo

https://pelis-react-cypz8z6sc-juanng94s-projects.vercel.app/

---

## Tecnologías utilizadas

- React
- JavaScript 
- Vite
- HTML5
- CSS3
- API REST (TMDB)
- Git & GitHub
- Deploy con Vercel

---

## Funcionalidades

- Listado de películas populares
- Búsqueda de películas por nombre
- Página de detalle de cada película
- Paginación
- Estados de carga con skeletons
- Manejo de errores y resultados vacíos
- Diseño responsive

---

## Qué demuestra este proyecto

- Consumo de APIs externas
- Manejo de estados en React (loading, error, empty)
- Uso de hooks personalizados
- Componentización
- Buenas prácticas de UI
- Flujo de trabajo con Git y GitHub
- Deploy de aplicaciones frontend
